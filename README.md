## Technical Details

- **Drupal Version:** Drupal CMS 1.2.8 (based on Drupal 11)
- **Installation Profile:** `drupal_cms_installer`
- **Site Title:** The GOAT Drupal CMS Site
- **Main Page:** Meet the Tubgoats (/tubgoats)

### Custom Content Type: Goat Profile

Fields:
- **Name** - The goat's name
- **Breed** (taxonomy reference) - Links to _Goat breeds_ vocabulary
- **Bio** (text, long) - Their story and achievements
- **Favorite Activity** (text) - What they love to do
- **Fave Tub Style** (text) - Their preferred bathing environment
- **Special Skills** (text) - What makes them unique
- **Featured Image** (media reference) - Professional headshot, generated by AI (ChatGPT/DALL-E)

## Local Development with DDEV

### Prerequisites

- [DDEV](https://ddev.readthedocs.io/) installed
- Docker running
- Composer installed

### Quick Start

1. **Clone this repository:**
   ```bash
   git clone https://github.com/YOUR-USERNAME/demo-tugboat-drupal-cms.git
   cd demo-tugboat-drupal-cms
   ```

2. **Start DDEV:**
   ```bash
   ddev start
   ```

3. **Install dependencies:**
   ```bash
   ddev composer install
   ```

4. **Import configuration and install the site:**
   ```bash
   ddev drush site:install --existing-config -y
   ```

5. **Visit your site:**
   ```bash
   ddev launch
   ```

6. **Login as admin:**
   ```bash
   ddev drush uli
   ```

## Configuration management

This project uses Drupal's configuration management system with the config directory at `/config` (outside the web root for security).

**Export configuration after changes:**
```bash
ddev drush config:export -y
```

**Import configuration:**
```bash
ddev drush config:import -y
```
